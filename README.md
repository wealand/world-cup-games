# 2026 FIFA World Cup Schedule

A premium, glassmorphic web dashboard that displays the schedule, locations, and match analysis for the 2026 FIFA World Cup games.

## Features
- **Modern Glassmorphic Design:** Visual aesthetics tailored withOutfit typography, smooth micro-animations, and vibrant color gradients.
- **Interactive Venue Filter:** Easily query matches scheduled across all cities and stadiums.
- **Robust Offline-First Fallback:** Bypasses browser CORS restrictions gracefully by falling back to a pre-fetched tournament data package if the live ESPN scoreboard API is blocked.
- **Buy Me a Coffee Integration:** Clean support footer linked to the developer profile.

## Getting Started
Simply open `index.html` in your favorite browser, or serve it locally using a simple HTTP server:
```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve
```
